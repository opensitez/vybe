# Console Demo

Demonstrates the full interactive console features of vybe Basic:

- **16 ConsoleColor values** — ForegroundColor / BackgroundColor
- **Colored output** — status messages with [OK], [WARN], [ERR] prefixes
- **Background highlighting** — colored badges (Info, Success, Warning, Error)
- **Console.ReadLine** — interactive text input
- **Number guessing game** — input loop with colored feedback
- **Console properties** — WindowWidth, WindowHeight, BufferWidth, BufferHeight
- **Console.ResetColor** — restore default colors

## Run in Editor

Open `ConsoleDemo.vbproj` in the vybe editor and press ▶ Run.

## Run from CLI

```bash
cargo run -- examples/console_demo/ConsoleDemo.vbproj
```
